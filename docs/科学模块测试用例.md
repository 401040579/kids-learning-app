## 1. 测试概述

### 1.1 测试范围

| 模块 | 功能点 |
|------|--------|
| 入口 | 首页科学探索卡片 |
| 主题选择 | 3个主题展示、进度显示 |
| 答题流程 | 题目展示、选项点击、答案判断 |
| 反馈弹窗 | 答对/答错反馈、知识拓展 |
| 积分系统 | 答题积分、完成奖励 |
| 数据持久化 | 进度保存、断点续学 |

### 1.2 测试环境

| 环境 | 配置 |
|------|------|
| 浏览器 | Chrome 90+, Safari 14+, Firefox 88+ |
| 移动端 | iOS Safari, Android Chrome |
| 屏幕尺寸 | 375px ~ 1920px |
| 网络 | 在线/离线 |

- [ ] 答错显示正确答案和解释
- [ ] 积分正确累加
- [ ] 进度正确保存和显示
